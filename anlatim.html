GÖRÜNTÜ İŞLEME

Matematiksel ve Sayısal Uygulama Kılavuzu

Bu sunumda piksellerin nasıl manipüle edildiğini, ham veriden (girdi) işlenmiş veriye (çıktı) dönüşümünü adım adım, gerçek matris değerleri üzerinden inceleyeceğiz.

01 Örnekleme (Sampling)

Örnekleme, görüntünün mekansal çözünürlüğünü azaltma işlemidir. INTER_NEAREST yönteminde, hedef koordinata en yakın olan orijinal piksel değeri doğrudan seçilir.

Adım s = 2 için (Yarı boyut):
Yeni[x, y] = Eski[x*2, y*2]
cv2.resize(img, (w//2, h//2), interpolation=cv2.INTER_NEAREST)

4x4 Girdi (Mavi Pikseller Seçiliyor)

150 30 200 45
20 80 100 10
40 15 255 12
90 55 0 200
↓

2x2 Çıktı Matrisi

150 200
40 255

02 Kuantalama (Quantization)

Kuantalama, piksellerin genlik (renk) derinliğini azaltır. Bit sayısı düştükçe renk geçişleri sertleşir.

Level = 2^bits (Seviye Sayısı)
Step = 256 / Level
Çıktı = (Girdi // Step) * Step

Örnek: 2-bit (4 renk seviyesi) için Step = 64. Her değer 0, 64, 128 veya 192'ye yuvarlanır.

8-bit Orijinal

45 120
180 250
→

2-bit Kuantalanmış

0 64
128 192

(45 // 64 = 0) * 64 = 0

05 Gaussian Filtre (Blur)

Gürültü azaltmak için kullanılır. Merkez piksel çevresindekilerle çarpılır ve ağırlıklı ortalama alınır.

Kernel (3x3):
1/16 * [[1, 2, 1], [2, 4, 2], [1, 2, 1]]
cv2.GaussianBlur(img, (3,3), 0)

Girdi (Merkez 200)

10 10 10
10 200 10
10 10 10
↓

İşlem Sonucu

(10*1 + 10*2 + 10*1 + ... + 200*4 + ...) / 16 = 57.5

... ... ...
... 58 ...
... ... ...

06 Morfoloji: Dilation (Genişletme)

Dilation, nesneleri kalınlaştırır. Kernel (3x3 birler matrisi) altındaki en büyük değeri (binary'de 1 varsa 1'i) seçer.

Y[x, y] = Max(Orijinal Pencere ∩ Kernel)
kernel = np.ones((3,3), np.uint8)
cv2.dilate(img, kernel, iterations=1)

Binary Girdi

0 0 0
0 1 0
0 0 0
→

Dilation Sonucu

1 1 1
1 1 1
1 1 1

08 Sobel Kenar Algılama

Sobel, pikseller arasındaki yoğunluk farkını (gradyan) hesaplar. Fark ne kadar büyükse kenar o kadar belirgindir.

Gx = [[-1, 0, 1], [-2, 0, 2], [-1, 0, 1]]
Gradyan (G) = |Gx| + |Gy| (Yaklaşık)
cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)

Dikey Kenar Örneği

10 10 200
10 10 200
10 10 200
↓ (Yatay Fark Gx)

Sobel Gradyan Çıktısı

0 255 0
0 255 0
0 255 0

Ortadaki büyük farklar tespit edildi.

11 Emboss (Kabartma)

Görüntüye 3D bir derinlik verir. Pikselleri bir yönden çıkarıp diğer yönden toplar. Fark yoksa 0 olur, bu yüzden 128 (nötr gri) eklenir.

Kernel:
[[-2, -1, 0],
 [-1,  1, 1],
 [ 0,  1, 2]] + 128 (Offset)

Düz Zemin (Piksel=50)

50 50 50
50 50 50
50 50 50
→

Emboss Çıktısı

128 128 128
128 128 128
128 128 128

Değişim yoksa sonuç nötr gridir.
