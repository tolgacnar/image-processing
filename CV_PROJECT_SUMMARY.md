## Project Title (suggestion)

## Bullet Points

- Built image enhancement and noise-restoration pipelines using histogram equalization, CLAHE, Gaussian blur, and median filtering.
- Developed morphological image operations (erosion, dilation, opening, closing) for shape-based image cleanup and analysis.
- Implemented multiple thresholding and edge-detection algorithms (Otsu, adaptive thresholding, Sobel, Prewitt, Laplacian, Canny) for image segmentation tasks.
- Built unsupervised image segmentation using K-Means clustering and connected-component (blob) labeling/analysis.
- Implemented geometric image transformations (rotation, scaling, affine and perspective warps).
- Integrated deep learning object detection and instance segmentation models (YOLOv8, YOLOv10) for real-time object recognition.
- Developed a real-time computer vision application that detects and tracks objects from a live webcam feed using YOLOv8 segmentation, supporting multiple simultaneous targets with custom on-screen visualization built with OpenCV.
- Applied HSV-based color filtering to improve detection robustness in real-time video processing.
- Documented algorithm derivations and mathematical formulas (sampling theorem, convolution kernels, filter equations) using LaTeX.
